# BugBot setup (one-time, before demo)

1. Open **Cursor Dashboard** → **BugBot** (or https://cursor.com/dashboard?tab=bugbot)
2. Install / configure the **Cursor BugBot** GitHub App for your account
3. Enable BugBot on repository **`Jstein77/excalidraw-demo`**
4. Confirm PR reviews run automatically on opened PRs

During the live demo, after opening the fix PR, wait ~30–60s and refresh the PR page to show BugBot's review comment.

If BugBot is not enabled, narrate: "BugBot would run here on the PR — same agent review loop your team gets in CI."
